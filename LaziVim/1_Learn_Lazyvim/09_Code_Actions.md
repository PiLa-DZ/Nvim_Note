# Code Actions

1. `Space c a` Code Action
1. `Space c r` Rename
1. `Space c s` Symbols
1. `Space c d` Line Diagnostics
1. `Space c l` LSP Info

## Action

| Shortcut | Action               | What it does                                                        |
| -------- | -------------------- | ------------------------------------------------------------------- |
| **gd**   | Go to Definition     | Jump to where the variable/function was first created.              |
| **gr**   | Go to References     | See every single place this function is being used in your project. |
| **gI**   | Go to Implementation | Jump to the actual code of an interface or abstract class.          |
| **K**    | Hover                | Show a popup with the documentation for the word under the cursor.  |
