# Keymap

## Edit Shortcut

```lua
-- ~/.config/nvim/lua/config/keymaps.lua
-- INFO: Normal Mode Esc or jj
vim.api.nvim_set_keymap("i", "jj", "<Esc>", { noremap = false })
```
