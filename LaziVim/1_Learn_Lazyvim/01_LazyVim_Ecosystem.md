# Lazivim Ecosystem

## Lazy.nvim (The Plugin Manager)

- Key Command: :Lazy
- Downloading and loading all the plugins
  (like the file explorer, the status-bar, etc.)

## Mason (The External Tool Manager)

- Key Command: :Mason
- Installs LSPs (Language Servers for autocompletion),
  DAPs (Debuggers), and Formatters (like Prettier for JS).

## LazyExtras (The Feature Library)

- Key Command: :LazyExtras
- Library of pre-configured settings built into LazyVim
- What it does:
  Instead of you having to write 50 lines of code
  to set up "TypeScript support"
  or "Docker support,"
  you just find it in this list and enable it.
