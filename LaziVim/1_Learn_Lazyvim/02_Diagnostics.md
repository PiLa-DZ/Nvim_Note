# Diagnostics

## Shortcut

1. `Space x x` Diagnostics (Trouble)
1. `Space u d` Diagnostics (UI)
1. `Space s d` Diagnostics (Search)
1. `Space c d` Diagnostics (Line)
1. `]d` Next Line (Diagnostics)
1. `[d` Previous Line (Diagnostics)

## What are Diagnostics?

Diagnostics come in four severity levels:

1. Error (Red):
   Your code will not run (e.g., a syntax error).

1. Warning (Yellow):
   Your code works,
   but something is likely wrong
   (e.g., a variable is defined but never used).

1. Information (Blue):
   General suggestions from the language server.

1. Hint (Cyan):
   Minor tips,
   like "this could be written more cleanly."
